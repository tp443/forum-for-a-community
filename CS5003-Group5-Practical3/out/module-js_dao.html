Module: js/dao

This is our Data Access Object. It defines functionality for storing information about Forum Posts and the Users who have signed up; they are stored in 2 seperate databases. The rest of our applications only needs to ask the DAO to store something in either of these databases. This file just conatines code regarding the two databses inplementation, The Week 9 complete app example by Dr Kasim Terzic proved a great base, and some of his code was paraphrased and implemented
